# Elementos de diseño

## Color
Debido al tema del evento del MALI se optó por utilizar una paleta de colores de blanco,negro y gris que permitan a los usuarios estar mas inmersos yb despertar ciertas emociones relacionadas al sentimiento de restriccion social 

## Space

Se trato de utilizar medidas proporcionales que le causen agrado al uauario por su simietria

## Size

Se jugó con los tamaños de las letras para causar mas impacto en cuanto a la lectura del usuario además de resaltar cierta importancia en alginos enunciados

# Arquitectura de la informacion

## Disclosure
Se mostró solo la informacion importante y suficiente que ayude al usuario a tener una idea del tema de la pagina web sin sobrecargarlo de datos innecesarios

##Focused navigation

El menu de navegacion es facil de entender y contiene todo lo necesario de la pagina para que el usuario reciba la informacion adecuada
